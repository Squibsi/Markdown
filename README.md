### HTML kodeeksempel for en navigation med 3 links
```
<html lang="da">
<head>
  <meta charset="UTF-8">
  <title>Eksempel på navigation</title>
  <!-- Link til eksternt stylesheet -->
  <link rel="stylesheet" href="css/style.css">
</head>
<body>

  <nav>
    <a href="#home">Hjem</a>
    <a href="#om-os">Om os</a>
    <a href="#kontakt">Kontakt</a>
  </nav>

</body>
</html>

```
